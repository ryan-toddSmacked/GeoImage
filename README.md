# GeoImage

A C++ library for working with GeoTIFF images using libtiff.

## Features

- Read and write GeoTIFF images
- Access geospatial metadata
- Simple and intuitive API

## Requirements

- C++17 or later
- CMake 3.14+
- libtiff (automatically fetched via CPM)

## Building

```bash
mkdir build
cd build
cmake ..
cmake --build .
```
